Project Compass

A futuristic project navigator & runner for modern polyglot development. Navigate, build, test, and manage your multi-stack workspace from one terminal UI.

Get Started npm

Why use Project Compass?

Everything you need to manage real-world, multi-stack development environments.

Automatic Discovery

Instantly identifies Node.js, Python, Rust, Go, Java, PHP, Ruby, .NET projects in any workspace.

AI Integration

OpenRouter, Gemini, Claude, Ollama support. Analyze projects, generate commands, get insights.

TUI + CLI Modes

Full terminal UI for interactive use plus complete non-interactive CLI for scripting and automation.

Native Package Mgmt

Automatically uses your project's preferred manager: uv/poetry/pipenv/pip, npm/pnpm/yarn/bun, cargo, go, maven/gradle, composer, bundler, dotnet.

Task Orchestration

Orbit Task Manager keeps background processes running, with live log streaming, kill/rename/detach support.

Project Scaffolding

8+ modern templates: Next.js, React (Vite), Vue, Rust, Django, FastAPI, Go, Spring Boot, Laravel, Rails, ASP.NET Core.

Get Installed

One command to rule your entire workspace.

Global install (recommended)

npm install -g project-compass

Install from source

git clone https://github.com/CrimsonDevil333333/project-compass.git
cd project-compass
npm install
npm start
